Jeu du Pion : Règles et Instructions

Fais par RAHMANI Rayane

Pour jouer:
Sur le terminal Shell de Linux:
utiliser la commande "make"
puis écrivez sur le terminal "./jeu"
enfin une fois le jeu terminez pour enlevez les fichiers ".o" effectuer la commande "make clean"

Objectif:

Déplacez le pion (En Rouge) sur la grille pour atteindre le trésor (En Jaune) sans tomber dans un piège (En Bleu).
Les murs (En vert) forment le contour de la grille et ne peuvent pas être traversés.

Contrôles :

Déplacement : Utilisez les flèches du clavier (↑, ↓, ←, →) pour déplacer le pion.

But : Atteindre la case jaune (le trésor) sans passer sur une case bleue (piège).

Perte : Si vous marchez sur une case bleue, vous perdez la partie ou tu dois recommencer le niveau.

Quitter : Appuie sur la touche Échap (Esc) pour quitter le niveau de la partie à tout moment.

Victoire :
Quand vous atteignez la case jaune (le trésor), appuie sur la flèche de gauche (←) pour valider votre victoire et passez au niveau suivant(ou finir le jeu).

Bug Possible:

Il se peut que selon le terminal linux le jeu ne s'affiche pas bien et laisse les bords pour accéder au trésor de la manière la plus facile.(cela depend de l'affichage de votre terminal)

Conseils :

Il existe toujours un chemin sûr pour atteindre le trésor dans chaque niveau.

Plus vous avancez dans les niveaux, plus il y a de pièges et la difficulté augmente.

Bon jeu !
